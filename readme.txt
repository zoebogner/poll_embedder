This plugin allows users to embed polls from PollDaddy.com (and eventually other services too) using Wordpress shortcode. The plugin doesn't make changes to user content, it simply intercepts content on its way to the view shows the embedded poll.

Code has been adapted from Ray J's Embed Extender plugin. The developer nor the plugin is affiliated with Polldaddy, Wordpress, Elgg, etc.

Supported polls
---------------
* Polldaddy Poll
* Polldaddy Rating
